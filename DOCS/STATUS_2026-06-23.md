# MVP Status - 2026-06-23

## Completed

- Repo/bootstrap docs are in place.
- Python package `zvook_doa` is implemented.
- SRP-PHAT coarse/refine localization works on synthetic 4-channel data.
- Config YAML loading is available through `ArrayConfig.from_yaml()`.
- CLI scripts support `--config`.
- WAV and realtime scripts support optional `--calibration`.
- MVP self-check script `scripts/check_mvp.py` runs synthetic matrix validation and benchmark.
- CLI scripts print clear `error: ...` messages for expected user-facing failures.
- Test geometry config `configs/test_array_microphone_class_4mic.yaml` matches the provided `MicrophoneArray` class.
- 4-channel FLAC is supported through `soundfile` via the same `run_wav.py` pipeline.
- Detector and refinement frequency analysis now use the `120-3000 Hz` band.
- Experiment configs support `distance_mics` and `analysis_band_hz` aliases.
- WAV/realtime pipelines support RAW SRP-PHAT output without `p_drone` or `confidence`.
- Detector stub, confidence scoring, tracker, WAV IO, simulation, and calibration helpers are implemented.
- Project-local Codex rules are committed and pushed.
